クライアントサイド PDF Tool

「PDFを開く」または「画像→PDF」で開始してください。

・サムネイルをドラッグして並べ替え（複数選択: Shift クリック=範囲 / Ctrl・Cmd クリック=個別トグル / Shift+←→=範囲拡張 / ←→=まとめて移動）
・サムネ右上の × で 1 ページ削除 / 複数選択中に Delete・Backspace キーでまとめて削除
・「↺左回転 / ↻右回転」で 1 ページずつ回転（サムネ未選択時は表示中の 1 枚、複数選択でまとめて回転）
・ツールバー右のズーム（−／100%／＋）で拡大縮小、フォントはプルダウンで切替
・ツールバーで注釈ツールを選択 → ページ上で描画
・「コメント」ツール: ページ上をクリック → 右外のガターに吹き出し。A− / A+ でサイズ、「返信」で返信追加、右下ハンドルで枠拡大
・全処理はブラウザ内で完結（外部送信なし）
