1 Getting Started — How to Launch

Unzip the downloaded ZIP file, then double-click the main file inside. The filename differs depending on your operating system:

■ Mac:
Start_Here.html

■ Windows:
pdf_tool_bundled.html

Double-clicking opens the file in your default browser (Chrome, Edge, Safari, Firefox, etc.) and the tool is ready to use immediately. No internet connection is required.

The Mac and Windows versions are identical in functionality. Every feature described in this guide — text, highlight, frame, comments, page reordering, zoom, font switching, etc. — works the same on both operating systems. The only difference is the filename you double-click to open.
If a dialog asks "Which app to use?", choose Chrome, Edge, or Safari. You do not need to open any other files (e.g., the lib folder). The single main HTML file is fully self-contained.
If the file won't open on Windows: Copy the unzipped folder to your desktop or Documents folder (not an external drive), then double-click pdf_tool_bundled.html again. If that still doesn't work, open Chrome or Edge first, then drag and drop pdf_tool_bundled.html onto the browser window.

2 Opening a PDF

Click the "Open PDF" button in the upper-left corner and select the PDF file you want to edit.

Alternatively, drag and drop a PDF file directly onto the browser window.

Once the PDF appears, you are ready to edit.

3 Choosing a Tool

The buttons along the top of the screen are your tools. Click one to activate it.

  • Select Select, move, resize, or edit annotations
  • Text Type text onto the PDF
  • Highlight Paint over text like a marker pen
  • Frame Draw a rectangular border
  • Strikethrough Cross out text with double lines
  • Pen Draw freehand lines with your mouse
  • Comment Add a margin callout outside the page (see the dedicated section below)
  • Eraser Click an annotation to delete it one at a time
To the right of the tool buttons you will find a color picker and a size input for adjusting the drawing color, stroke width, and font size.
Font selection: Use the font dropdown on the right side of the toolbar to choose the font for text annotations, comments, and page numbers. The default is "Gothic (Sawarabi Gothic)". To use your own font, select Load custom font… at the bottom of the dropdown and pick a .ttf or .otf file. Once loaded, the font stays in the menu and can be switched at any time.
Zoom: Use the − / 100% / + buttons at the far right of the toolbar to adjust the display scale. Click 100% to reset to the default size.
Language toggle: The JP / EN button at the rightmost end of the toolbar switches the interface language between Japanese and English. Your preference is saved in the browser and persists across sessions.

4 Adding Text

  1. Click the "Text" tool to activate it.
  2. Click on the PDF where you want to place text.
  3. An input box appears — type your text.
  4. Press Enter to confirm the input.
  5. Press Enter once more to finalize the text annotation.
If you are composing with an IME (e.g., typing Japanese), the first Enter key confirms the IME conversion; the second Enter finalizes the annotation. Long text automatically wraps to multiple lines to fit the box width.
To re-edit existing text, double-click the annotation to enter edit mode again.

5 Highlight, Frame, and Strikethrough

  1. Select "Highlight", "Frame", or "Strikethrough".
  2. Drag (click and hold, then move) over the area on the PDF where you want to place the annotation.
  3. Release the mouse button — the annotation is created and immediately selected (handles appear at the corners).

To pick a color in advance, click the color button in the toolbar before drawing.

To change color or size after creation, switch to the Select tool and click the annotation. Frames and highlights can be clicked anywhere inside their bounding area — not just on the border line.

When you are done editing, click a blank area of the page or press Enter to deselect and finalize.

Adjusting Strikethrough Lines

The strikethrough annotation uses two parallel lines to indicate deletion. When you select a strikethrough, three types of fill handles appear:

  • Blue square handles (both ends): Drag each end individually to reposition the start or end point — useful for changing length and angle.
  • Orange handle (center): Drag along the line direction to stretch or shrink the line symmetrically about its midpoint.
  • Green handle (beside the line): Drag perpendicular to the line to widen or narrow the gap between the two lines — handy when matching large-font headings.

You can also adjust values numerically in the properties panel (top-right). "Stroke width" sets the thickness of each individual line; "Double-line gap" sets the distance between the two lines (in pt).

Tip for large text: Set "Stroke width" to 1.5–3 pt and "Double-line gap" to 4–10 pt for a bold strikethrough that stands out on large headings.

6 Freehand Drawing

  1. Select the "Pen" tool.
  2. Drag your mouse across the PDF to draw freely.

Adjust the stroke width using the "Size" field in the toolbar. Color can also be changed with the color button.

7 Comments (Margin Callouts)

Like the comment feature in Word, you can attach a note outside the page boundary, connected to the relevant area by a leader line.

  1. Select "Comment" from the toolbar.
  2. Drag over the area on the page you want to annotate (drag to select a region, or just click for a point anchor).
  3. When you release the mouse, a dashed border appears around your selection and a yellow callout box appears to the right of the page.
  4. Click the body area of the callout and type your comment.
  5. Drag the title bar to reposition the callout.
  6. Drag the small square handle at the bottom-right of the callout to resize it.

Adding Replies

Click the "Reply" button in the callout's title bar to add a child card below the parent comment. Replies are labeled "↳ Reply 1", "↳ Reply 2", etc., and are visually distinct from the parent.

  • Click the body area of a reply card to type its text
  • Press "Reply" repeatedly to add as many replies as needed
  • Delete an individual reply with the × button at its top-right
  • A vertical line from the parent and horizontal branches to each reply card show the thread structure at a glance

Changing Font Size

Use the A− / A+ buttons in the title bar to adjust the font size of that callout. Each callout can be sized independently; reply cards follow the parent callout's size automatically.

Deleting a Comment

Click the × at the far-right of the parent callout's title bar to delete the entire comment thread, including all reply cards.

How saving works: When you export the PDF via "Save & Download", pages that have comments automatically get a right-side margin area added so the callouts are embedded at the same positions you see on screen. The original page content is never hidden or overlapped.
As a side effect, pages with comments will be wider than the original PDF. Keep this in mind when printing (e.g., an A4 PDF with comments becomes wider than A4 on those pages). Pages without comments retain their original width.

8 Editing and Deleting Annotations

Selecting

Switch to the "Select" tool and click on an annotation. For frames and highlights, you can click anywhere inside the bounding area (not just on the border line itself). A blue dashed outline with blue square fill handles at the corners and edges indicates that the annotation is selected.

Moving

Drag from inside the selection to reposition the annotation.

Resizing

Drag the fill handles at the corners or edges to change width or height.

  • Frame / Highlight: 8 handles (4 corners + 4 edges) for free resizing in any direction
  • Text: 6 handles (4 corners + 2 side edges) to adjust width; height adjusts automatically to fit the text

Changing Color, Stroke Width, or Font Size

When an annotation is selected, a properties panel appears in the upper-right of the screen. Use it to adjust color, stroke width, font size, opacity, width, and height.

How to deselect:
To dismiss the selection handles, use any of the following:
  • Click a blank area of the page
  • Press Enter
  • Press Escape
  • Switch to a different tool

Deleting

  • Click the red "Delete" button in the properties panel (recommended)
  • Switch to the "Eraser" tool and click the annotation
  • For comments, click the × button at the top-right of the callout

9 Reordering, Multi-selecting, and Deleting Pages

The thumbnail strip at the bottom of the screen shows small previews of each page. Drag thumbnails left or right to reorder pages.

Click the × button at the top-right of a thumbnail to delete that page.

Selecting Multiple Pages at Once

You can select multiple pages to reorder or delete them together:

  • Click: Selects only the clicked page (previous selection cleared)
  • Shift + Click: Selects all pages from the previously selected page to the clicked page
  • Ctrl + Click (Mac: Command + Click): Toggles individual pages in or out of the selection
  • Shift + ← / →: Extends the current selection one page at a time in the arrow direction (does not change page order)
  • ← / → (without Shift): Moves all selected pages one step left or right
  • Drag: Drag the selected thumbnails to move them all to a new position (a badge showing "N" pages appears)
  • Delete / Backspace: Deletes all selected pages at once (a confirmation dialog appears)
Holding Shift only extends the selection — it does not move pages. To reorder pages, release Shift and use the arrow keys or drag.
Bulk delete: Ctrl/Shift-click to select the pages you want to remove, then press Delete (Mac: delete / fn+delete) or Backspace. All selected pages are removed at once. A confirmation dialog appears before deletion.

10 Rotating Pages 90°

Use the "↺ Rotate Left" and "↻ Rotate Right" buttons in the toolbar to rotate pages by 90° increments. This is useful when scanned PDFs are imported sideways.

  1. Select the page(s) you want to rotate in the thumbnail strip (multi-select is supported).
  2. Click "↺ Rotate Left" or "↻ Rotate Right". All selected pages rotate together by 90°.
If no thumbnail is selected, only the page currently visible in the main view is rotated. To fix pages one by one, scroll to the target page and then click the rotate button.

Rotation can be repeated as many times as needed (4 clicks returns to the original orientation). The rotated orientation is preserved when you save.

11 Saving the Edited PDF

Click the blue "Save & Download" button in the upper-right to download the edited PDF to your computer (it goes to the browser's Downloads folder).

The saved filename appends _edited to the original name:

Example: report.pdf → report_edited.pdf

Annotation sizes are not affected by zoom level. Even if you write text or highlights while zoomed out, the positions and sizes will look correct at full size and when printed. Font sizes and stroke widths are stored as PDF points (pt) — independent of the display zoom.

12 Other Useful Features

Convert Images to PDF

Click the "Image → PDF" button to select PNG or JPEG files and combine them into a single PDF.

Merge Multiple PDFs

Click the "Add PDF" button to append another PDF file to the end of the currently open document.

Insert Page Numbers

Click the "Page Numbers" button to open a dialog. Choose the position (e.g., bottom center) and format (e.g., 1/10), then click "Add" to automatically insert page numbers on every page.

Start Over

Click the "Clear" button to discard the current PDF and start fresh.

13 Troubleshooting

I typed text but it disappeared

If you are composing with an IME, you may have pressed Enter twice in quick succession — once to confirm the IME conversion and immediately again to finalize the annotation, which deleted it. Confirm the conversion first, then press Enter a second time to place the annotation.

Buttons do not respond when clicked

Try updating your browser to the latest version. Recommended browsers are the latest versions of Chrome, Edge, Safari, or Firefox.

The font looks wrong or characters are not displaying correctly

This app ships with a built-in font (Sawarabi Gothic), selected by default in the font dropdown. To use a different font, choose Load custom font… at the bottom of the dropdown and load a .ttf or .otf file. Once loaded, it stays in the menu and can be reselected at any time.

I can't select a frame annotation by clicking on it

Make sure the Select tool is active, then click anywhere inside the frame — not just on the border line itself. Clicking the central area works fine.

How do I deselect an annotation after editing it?

Click a blank area of the page, or press the Enter or Escape key. The selection handles will disappear and the annotation will be finalized.

Is there any difference between the Mac and Windows versions?

No. Both versions are identical in content and functionality. This guide applies to both operating systems. The only difference is the filename you double-click to launch the tool (Mac: Start_Here.html; Windows: pdf_tool_bundled.html).

Opening a large PDF is slow

PDFs with more than 100 pages may take some time to load. Please wait while the file is being processed.

I'm worried that my files might be sent to an external server

This app sends nothing to any external server. There is no code in the source to upload your images, PDFs, or typed text anywhere. You can turn off Wi-Fi while using it and it will continue to work perfectly. Your data stays entirely on your own computer.

14 About This App

This tool is built using a "client-side" architecture. In plain terms:

All PDF editing happens entirely inside your browser.

It works without an internet connection, and no files or text you type ever leave your computer.

Confidential business documents, files containing personal information, medical records — any file you cannot afford to upload externally can be edited safely with this tool.
